### **4. Model Selection and Evaluation**
For each model:
- **Parameter Tuning**: Adjust parameters (e.g., α, β, γ for exponential smoothing; p, d, q for ARIMA).
- **Cross-Validation**: Apply time-series cross-validation.
- **Performance Metrics**: Use RMSE, MAPE, or other suitable error metrics.

### **Additional Techniques Covered**
- **Moving Averages**: For trend or seasonal component smoothing.
- **Differencing**: To make non-stationary data stationary.
- **Decomposition**: Separate components (trend, seasonal, residual) for exploratory analysis.
